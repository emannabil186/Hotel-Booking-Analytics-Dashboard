# Hotel Booking Analytics Dashboard

An interactive Business Intelligence dashboard developed in Microsoft Excel to analyze hotel reservation behaviors, seasonal booking trends, and operational metrics to support data-driven decision-making.

## Key Features & Analyses
* **Booking Trends:** Analyzed customer behaviors, lead times, and seasonal variations to optimize revenue management.
* **Cancellation Analysis:** Built insights to track cancellation rates and optimize occupancy levels.
* **Interactive Visualizations:** Integrated dynamic filters, slicers, and advanced Excel charts for seamless data exploration.
